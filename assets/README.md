# Assets Directory

This directory contains screenshots and visual assets for documentation.

## Screenshots to Add

1. **`chart1_preview.png`** - Screenshot of top 10 PM2.5 cities bar chart
2. **`chart2_preview.png`** - Screenshot of monthly NO₂ trend line chart
3. **`chart3_preview.png`** - Screenshot of PM2.5 vs NO₂ scatter plot with regression
4. **`pipeline_output.png`** - Terminal screenshot showing pipeline execution
5. **`notebook_demo.png`** - Screenshot of Jupyter notebook in action

## How to Create Screenshots

### From HTML Charts

1. Open HTML file in browser
2. Take screenshot using:
   - **Mac**: Cmd + Shift + 4
   - **Windows**: Windows + Shift + S
   - **Linux**: Screenshot tool

3. Save as PNG with descriptive filename

### From Terminal/Notebook

1. Run pipeline or notebook
2. Capture output window
3. Crop to relevant section
4. Save as PNG

## Usage in README

Screenshots can be embedded in the main README using:

```markdown
![Chart Preview](assets/chart1_preview.png)
```

## Recommended Dimensions

- Charts: 1200x600px (for GitHub preview)
- Terminal: 800x400px
- Notebook: 1400x800px
